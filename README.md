## О себе
**Full-Stack .NET / React разработчик**.

Разрабатываю внутренние веб-приложения для автоматизации бизнес-процессов: от сбора требований и проектирования архитектуры до backend, frontend, деплоя и сопровождения.

Основной стек — **C#**, **ASP.NET Core**, **Entity Framework Core**, **PostgreSQL**, **React** и **TypeScript**. **Rust** как дополнительный язык.
